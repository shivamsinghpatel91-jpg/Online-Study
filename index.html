Engineering Study Path – After Class 10th & 12th

Qualification Possible Courses / Streams Duration Next Step / Future Path
Class 10th ITI Courses (Electrician, Fitter, Computer, Mechanical) 1-2 Years Diploma / Polytechnic → B.Tech / B.E
Class 10th Diploma in Engineering (Mechanical, Civil, Electrical, Electronics, Computer) 3 Years Lateral Entry in B.Tech / B.E (2nd Year)
Class 10th Polytechnic / Technical Certificate Courses 2-3 Years B.Tech / B.E (via Lateral Entry)
